org.apache.uima.util

Interface SaxDeserializer

  • All Superinterfaces:
    ContentHandler

    public interface SaxDeserializer
extends ContentHandler
    Deserializes an XMLizable object from SAX events. This interface extends the SAX ContentHandler interface and through this interface can receive events generated by a SAX parser. When the parse has completed, the getObject() method may be called to retrieve the XMLizable object.
