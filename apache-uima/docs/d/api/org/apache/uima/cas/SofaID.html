org.apache.uima.cas

Interface SofaID

  • Deprecated. 
    As of v2.0, this is no longer needed. CAS views can now be obtained directly using String identifiers, through the method CAS.getView(String).

    @Deprecated
public interface SofaID
    Interface that contains the handle to a Sofa FS in the CAS. The SofaID is obtained from the UimaContext by calling UimaContext.mapToSofaID().
