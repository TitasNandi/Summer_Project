Interface IndexBuildItem

  • All Superinterfaces:
    Serializable, XMLizable

    public interface IndexBuildItem
extends XMLizable, Serializable
    An individual item within a IndexBuildSpecification. Specifies the getName() name of a CAS type, a index rule, and an optional filter.

    This object implements the XMLizable interface and can be parsed from an XML representation.
