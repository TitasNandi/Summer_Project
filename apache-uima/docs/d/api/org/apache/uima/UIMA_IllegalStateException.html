    Serializable

    public class UIMA_IllegalStateException
extends UIMARuntimeException
    Signals that a method has been invoked at an illegal or inappropriate time. In other words, the object on which the method was called is not in an appropriate state for the requested operation. This extends RuntimeException and so does not need to be declared in the throws clause of methods.
